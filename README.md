# Progenic

Multi-workers daemon module


## Getting Started

Install the module with: `npm install progenic`

```js
var progenic = require('progenic');
var numberOfWorkers = 4;
var devMode         = false;
progenic.run('myServiceName', 'path/to/myServiceScript.js', numberOfWorkers, devMode);
```
